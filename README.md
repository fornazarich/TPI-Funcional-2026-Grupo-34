# TPI-Funcional-2026-Grupo-34

##  Objetivos del Proyecto

El presente Trabajo Práctico Integrador tiene como finalidad aplicar los fundamentos del paradigma funcional para la resolución de un problema complejo de ingeniería. Se persiguen los siguientes objetivos clave:

* ** Modelar un problema del mundo real**
    Traducir los requisitos funcionales de un sistema de tráfico urbano a código **LISP**, abstrayendo la lógica del dominio mediante funciones.
    
* ** Aplicar conceptos fundamentales**
    Implementar soluciones basadas en *funciones puras*, *inmutabilidad* y *composición funcional* para estructurar un software robusto y predecible.
    
* ** Desarrollar pensamiento algorítmico**
    Diseñar soluciones elegantes y eficientes que reflejen la naturaleza matemática y la expresividad de LISP.
    
* ** Exposición a escenarios desafiantes**
    Enfrentar circunstancias nuevas y desconocidas, potenciando la capacidad de adaptación técnica y la investigación autónoma.
